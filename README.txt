We analyzed the Cyclistic bike-share data made available by Motivate International Inc. under this (https://ride.divvybikes.com/data-license-agreement) license.

import-data.R - downloads and imports csv file of 12 months of 2021-Cyclistic trip data to data directory on local pc, from: https://divvy-tripdata.s3.amazonaws.com/index.html

wrangle-data.R - creates a derived dataset and saves as R object in rda directory

analyze-data.R - generates various analysis plots and saves it to the cyclistic_report_files/figure-gfm directory
